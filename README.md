# Bernard Andy - Professional Portfolio

This repository contains the source code for my personal academic and professional website, hosted at [bernardandy.github.io](https://bernardandy.github.io).

## About Me
I am a Forensic Scientist and Policy Researcher specializing in Genomic Data Governance, Ethics, and Legislative Reform in Africa. I hold an MSc in Molecular Biology and Genomics and am currently pursuing advanced research in forensic DNA analysis.

## Site Structure
- **_pages/**: Contains the main pages (About, CV, etc.).
- **_posts/**: Contains blog posts and policy analyses.
- **_portfolio/**: Contains detailed policy briefs and case studies.
- **_config.yml**: The main configuration file for the site.

## Technology Stack
- **Platform**: GitHub Pages
- **Framework**: Jekyll
- **Template**: AcademicPages (a fork of Minimal Mistakes)
